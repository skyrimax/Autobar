Ce dossier contient les scripts permettant de créer la base de données, faire des exemples d'insertions de données et faire des exemples de requetes.

L'ordre d'éxécution doit être la suivante :
1-creation		(Fichier créant les tables)
2-creation_autres	(Fichier créant les objets de la base de données)
3-insertions	(Insertions de quelques examples de tuples de données)
4-Requete		(Examples de requetes intéressantes)
